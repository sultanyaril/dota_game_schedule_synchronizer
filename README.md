# Dota Game Schedule Synchronizer

A small utility to fetch and synchronize Dota game schedules.

## Contents
- `main.py` — the main script to run the synchronizer.
- `requirements.txt` — Python dependencies.

## Features
- Fetch upcoming Dota game schedules from configured sources.
- Normalize and output schedule data for use in other tools or notifications.

## Prerequisites
- Python 3.8+ installed on your system.
- (Optional) A virtual environment for isolated dependencies.

## Installation
1. Clone the repository or download the project files.
2. (Recommended) Create and activate a virtual environment:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

3. Install dependencies:

```powershell
pip install -r requirements.txt
```

## Usage
Run the main script:

```powershell
python main.py
```

Options and configuration may be available inside `main.py` — open it to see runtime arguments and configuration points.
